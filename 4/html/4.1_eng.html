4.1 Ориентированные ациклические графы 4.1 directed acyclic graph

В этом разделе изучаются свойства важного класса ориентированных графов, а именно ациклических . In this section we study the properties of an important class of directed graphs, namely acyclic. Как мы знаем, ориентированный граф — ациклический, если он не содержит контуров. As we know, a directed graph - is acyclic if it contains no circuits. Очевидно, простейшим примером ациклического ориентированного графа является ориентированное дерево. Obviously, the simplest example of an acyclic directed graph is a directed tree.

Основной результат, который мы получим в этом разделе, заключается в том, что вершины ациклического ориентированного графа G на n вершинах можно пометить таким образом целыми числами из множества {1, 2, ... , n }, что если в графе G имеется дуга (i, j ), то i < j . The main result that we obtain in this section is that the vertices of an acyclic directed graph G on n vertices can be marked so that integers in the set {1, 2, ..., n}, that if a graph G has an arc (i, j), then i <j.

Определенное нами упорядочение вершин называется топологической сортировкой. We defined the ordering of vertices is called a topological sorting.
